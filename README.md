This project is a simple ASP.NET Core MVC web application connected to a SQLite database that manages a sports event calendar. Users can filter events by sport and by date range, view event details one by one with easy previous/next navigation, and create new records through straightforward forms: add events (date, time, sport, place, teams), add sports, add places, and add teams.

To set up and run the app, install Visual Studio 2022 and the latest .NET SDK. Open the solution by double-clicking Sport_Calendar.sln then ensure the NuGet packages are installed: Microsoft.EntityFrameworkCore , Microsoft.Data.Sqlite.Core, and Microsoft.EntityFrameworkCore.Tools; and in the Solution Explorer right-click the Sport_Calendar project and choose Build; once it builds successfully, press F5 to run the application, making sure that port 7160 is free on your machine . To try the tests in the explorer solution Explorer right-click the Sport_Calendar.Test and run tests then wait.
